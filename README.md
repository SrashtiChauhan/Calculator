# 📌 Overview

This project is a responsive calculator application built using **HTML5**, **CSS3**, and **Vanilla JavaScript** without any external frameworks or libraries.

The project focuses on writing clean, maintainable code while following modern frontend development practices such as semantic HTML, separation of concerns, modular JavaScript functions, and responsive UI design.

# ⚙️ How It Works

The calculator maintains its current state using four variables.

```javascript
currentInput
previousInput
operator
shouldResetDisplay
```

These variables help the calculator behave like a real calculator while keeping the code clean and easy to maintain.
